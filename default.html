网站建设

高效、安全、稳定的建站方案，适用于企业产品展示站或个人博客站
高效：服务器只需处理静态文件，大大减少传输时间、增加最大同时访问人数。 安全：服务器上没有后台文件，全部为非执行文件，避免安全隐患。 稳定：使用版本控制软件，可以在不同电脑进行备份。
使用 Jekyll 快速建站 使用 Liquid 模板语言快速建立一个常规网站。如果功能不够也可以另外用 Ruby 写插件辅助。
瞬间完成建站命令 在本地执行建站命令，瞬间生成所有静态文件。当然也可以在服务器上安装 Jekyll 软件直接生成。
获得全站静态文件 网站文件一目了然，避免出现重要文件被抓取。经过代码处理，网站也可以在本地直接浏览。
通过版本控制软件 Git 管理 每次更新网站，提交更改，只需上传更改过的部分，毋须全站上传。版本控制记录每次更改历史，你可以通过命令恢复到之前任何一个版本。
自动发布 服务器配置 Git 库，经过第一次设置后，只要以后每次在本地电脑完成提交更改，服务器就会自动更新。
