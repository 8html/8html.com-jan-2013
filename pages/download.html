{% capture content %} ** 安装 git ** ``sudo apt-get install git`` ** 下载网站 ** ``git clone git://github.com/caiguanhao/8html.com.git`` ``cd 8html.com`` ``git submodule init`` ``git submodule update`` ** 安装需要的软件 ** ``sudo apt-get install gcc make libssl-dev`` ** 安装最新版 Ruby ** ``wget ftp://ftp.ruby-lang.org/pub/ruby/2.0/ruby-2.0.0-p0.tar.gz`` ``tar xfvz ruby-2.0.0-p0.tar.gz`` ``cd ruby-2.0.0-p0`` ``./configure`` ``make`` ``sudo make install`` ** 安装 Jekyll ** ``sudo gem install -V jekyll ruby-nuggets rdiscount`` ** 最后 ** 在 8html.com 目录下执行 jekyll 命令即可生成网站，网站内容存放在新建的 ``_site`` 目录下，你可以直接在浏览器中打开 ``_site `` 下的 index.html ，或者配置本地服务器软件，在局域网中查看。 ** 其他 ** pygments.rb 需要 Python 2.6 / 2.7 ，否则会出现错误。如果系统内没有 Python 请安装： ``sudo apt-get install python2.6`` ``sudo ln -s /usr/bin/python2.6 /usr/bin/python`` {% endcapture %} {{ content | markdownify }}
{% capture content %} ** 添加 EPEL YUM 源 ** *** CentOS 6: *** ``rpm -ivh http://dl.fedoraproject.org/pub/epel/6/i386/epel-release-6-8.noarch.rpm`` ``yum makecache`` *** CentOS 5: *** ``rpm -ivh http://dl.fedoraproject.org/pub/epel/5/i386/epel-release-5-4.noarch.rpm`` ``yum makecache`` ** 安装 Git 并下载网站 ** ``yum -y install git`` ``git clone git://github.com/caiguanhao/8html.com.git`` ``cd 8html.com`` ``git submodule init`` ``git submodule update`` ** 安装 Ruby ** ``yum -y install gcc make openssl-devel`` ``wget ftp://ftp.ruby-lang.org/pub/ruby/2.0/ruby-2.0.0-p0.tar.gz`` ``tar xfvz ruby-2.0.0-p0.tar.gz`` ``cd ruby-2.0.0-p0`` ``./configure`` ``make`` ``make install`` ** 安装 Jekyll ** ``gem install -V jekyll ruby-nuggets rdiscount`` {% endcapture %} {{ content | markdownify }}
{% capture content %} ** 更新 pacman ** ``haveged -w 1024`` ``pacman-key --init`` ``pkill haveged`` ``pacman-key --populate archlinux`` ``pacman -Syu`` ** 安装 Git 并下载网站 ** ``pacman -S git`` ``git clone git://github.com/caiguanhao/8html.com.git`` ``cd 8html.com`` ``git submodule init`` ``git submodule update`` ** 安装 Ruby ** ``pacman -S gcc make openssl`` ``curl -O ftp://ftp.ruby-lang.org/pub/ruby/2.0/ruby-2.0.0-p0.tar.gz`` ``tar xfvz ruby-2.0.0-p0.tar.gz`` ``cd ruby-2.0.0-p0`` ``./configure`` ``make`` ``make install`` ** 安装 Jekyll ** ``gem install -V jekyll ruby-nuggets rdiscount`` ** 安装 Python ** ``pacman -S python2`` ``ln -s /usr/bin/python2.7 /usr/bin/python`` {% endcapture %} {{ content | markdownify }}
